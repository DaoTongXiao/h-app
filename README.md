# h-app

### 启动

安装

```shell
pnpm i
```

运行，以微信小程序为例

```shell
pnpm run dev:mp-weixin
```

然后打开 `微信开发者工具`，导入 `dist/dev/mp-weixin` 项目，然后就可以预览了
